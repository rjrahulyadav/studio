# **App Name**: DhaniHya Solutions

## Core Features:

- Animated Hero Section: Visually engaging hero section with animated background and neon glow to immediately capture user attention.
- Interactive Service Cards: Dynamic service cards with icon animations, descriptions, and glowing hover effects to showcase the company's offerings.
- Real-Time Project Portfolio: Interactive project portfolio with filters and animations, highlighting real-time project experience.
- Dynamic Contact Form: Animated contact form with validation and integrated Google Maps for easy user interaction.
- AI-Powered Chatbot: A support chatbot that uses a retrieval tool to offer a personal user support experience.
- Training Module Registration: Enable users to view and register for training modules.

## Style Guidelines:

- Primary color: Electric Purple (#BE29EC) to create a modern, futuristic vibe.
- Background color: Dark Charcoal (#1A1A1A) provides contrast and a modern feel.
- Accent color: Neon Green (#39FF14) to highlight interactive elements and calls to action.
- Headline font: 'Space Grotesk' sans-serif for a modern, techy feel.
- Body font: 'Inter' sans-serif for clear, readable content.
- Use minimalist, glowing line icons to represent services and key features.
- Implement glassmorphism elements and parallax scrolling for a futuristic UI.
- Subtle scroll-based animations and hover effects to enhance user engagement.
